QA Projects – SauceDemo & WeatherMap API
📌 Project Overview
This repository contains my diploma project for Software Testing (SWT 513).
It demonstrates Manual UI Testing on SauceDemo and API Testing using the WeatherMap API with Postman.

🛠 Tools Used
Excel → Test case design & bug tracking
Postman → API testing & assertions
PowerPoint → Documentation & presentation of API results
GitHub → Portfolio hosting
✅ Manual Testing (SauceDemo)
Designed and executed 40+ test cases covering:
Login (valid/invalid credentials, empty fields, password masking)
Inventory (product listing, sorting, filtering, add-to-cart)
Cart (add/remove items, quantity, price calculation)
Checkout (mandatory fields, order completion, error handling)
Documented 15+ bugs with severity & priority classification.
Linked bug reports to test case IDs for traceability.
🌐 API Testing (WeatherMap API)
Created global variables (appid, lang, lon, lat) in Postman.
Tested API requests by city name and with multilingual support.
Validated responses using assertions:
Status Code = 200
Response Time < 250ms
JSON fields (City name, Country, ID)
Example:
City = القاهرة
Country = EG
All assertions passed successfully.

/BugReports → Excel files for bug tracking
/API_Testing → Postman collection + PowerPoint documentation
