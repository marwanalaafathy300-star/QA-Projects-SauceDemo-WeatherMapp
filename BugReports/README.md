# Bug Reports – SauceDemo

This folder contains the **bug tracking sheet** for the SauceDemo application.  
Each bug report includes:

- Bug ID  
- Linked Test Case ID  
- Description of the defect  
- Severity and Priority classification  
- Steps to reproduce  
- Expected vs. Actual results  
- Status (Open/Closed/Fixed)  

This ensures full traceability between test cases and defects.
